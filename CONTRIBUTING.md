# Contributing

## Setup

```bash
cd /Users/bryanjackson/Documents/code/CookbookClub
npm install
npm test
```

Node.js 24+ is required.

## Development Rules

- Keep changes focused and small.
- Run `npm test` before opening a pull request.
- Update docs when commands or behavior change.
- Prefer additive migrations and repair-safe storage changes.

## Branches

- Create feature branches with the `codex/` prefix.
- Open pull requests into `main`.

## Pull Requests

- Include a short summary of behavior changes.
- List verification steps and key command outputs.
- Call out data/model or migration impacts explicitly.
