# Security Policy

## Supported Versions

This project is currently pre-1.0 and maintained on the `main` branch.
Security fixes are shipped on `main`.

## Reporting a Vulnerability

Please do not open public issues for suspected vulnerabilities.

Send a report to:

- GitHub: open a private security advisory in this repository.

Include:

- Impact summary
- Steps to reproduce
- Affected commands/paths (for example `data import`, `sqlite doctor`)
- Suggested fix (if available)

We will acknowledge reports and work on a fix as quickly as possible.
